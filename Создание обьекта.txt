                          Создание объекта

Объекты являются экземплярами классов и представляют конкретные данные и поведение. 
Для создания объекта применяется конструктор. Конструктор - это специальный метод, который вызывается при создании нового объекта класса и выполняет инициализацию объекта. Имя конструктора должно совпадать с именем класса, в сигнатуре конструктора не указывается тип возвращаемого значения.

class Person
{
public string name = “Вася”;
public Person()	 // конструктор класса
{}
}

Если в классе не определено ни одного конструктора, то для этого класса автоматически создается пустой конструктор по умолчанию. Конструкторов может быть несколько, они должны отличаться по количеству и типу принимаемых аргументов.
Общий синтаксис вызова конструктора:

new  конструктор_класса (параметры_конструктора);

Сначала идет оператор new,  который выделяет память для объекта, а после него идет вызов конструктора.
Создание объекта класса:

Имя_класса имя_обьекта = new конструктор_класса (параметры_конструктора);

Пример создания объекта person класса Person:

Person person = new Person();

Для инициализации полей класса используется конструктор с параметрами.

class Person
{
private string name;
public Person(string xname)
{
name=xname;
}
}

Имя формального параметра конструктор может совпадать с именем поля, для того, чтобы их различать, у поля класса записывается служебное слово this, которое представляет ссылку на текущий экземпляр класса

class Person
{
private string name;
public Person(string xname)
{this.name=name; }
}

Для обращения к полям и методам класса применяется составное имя: после имени объекта ставится точка, а затем элемент класса:
имя_объекта. имя_метода(фактические параметры)

Прямой доступ к полям класса нежелателен. Доступ к полям осуществляется либо через методы класса, либо через свойства класса.
