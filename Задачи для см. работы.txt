                          Задачи для см. работы

Задача №1. Создать класс Студент, содержащий имя, возраст и средний балл. Предусмотреть в классе конструктор для инициализации полей класса и метод для вывода информации о студенте.

Задача 2: Наследование классов "Фигура" и "Треугольник"1. Создайте базовый класс Shape с методом для вычисления площади фигуры.
2. Создайте дочерний класс Triangle, который наследуется от Shape и реализует метод для вычисления площади треугольника.
3. Создайте объекты классов Shape и Triangle, вызовите методы для расчета площади.

Задача 3: Использование интерфейсов
1. Создайте интерфейс IMovable с методом Move().
2. Реализуйте интерфейс в классах Car и Plane, добавив свою реализацию метода Move().
3. Создайте объекты каждого класса, вызовите метод Move() для каждого.

Задача 4: Абстрактный класс "Животное"
1. Создайте абстрактный класс Animal с абстрактным методом MakeSound().
2. Создайте дочерние классы Dog и Cat, которые наследуются от Animal и реализуют метод MakeSound().
3. Создайте объекты каждого класса, вызовите метод MakeSound() для каждого животного.

Задача 5: Использование полиморфизма
1. Создайте базовый класс Shape с методом для вычисления площади.
2. Создайте дочерние классы Rectangle, Circle и Triangle, которые наследуются от Shape и реализуют метод для расчета площади соответствующей фигуры.
3. Создайте массив объектов различных фигур, вызовите методы для расчета площади и выведите результаты.
